Full-stack O2C (Order-to-Cash) system with SAP Fiori UI, Flask backend, SQLite database, invoice PDF generation, and real-time analytics dashboard.
# O2C SAP Project
This project demonstrates Order-to-Cash process using SAP Fiori style UI with Flask backend.

- Delivery, Billing, Payment
- Invoice PDF
- Dashboard charts
